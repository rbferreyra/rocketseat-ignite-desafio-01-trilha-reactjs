### Executar aplicação

1 - Clonar repositório

```bash
git clone https://github.com/rbferreyra/rocketseat-ignite-desafio-01-trilha-reactjs
```

2 - Instalar dependências

```bash
yarn install
```

3 - Abrir a aplicação

```bash
yarn dev
```

Visualizar a aplicação pelo [http://localhost:8080/](http://localhost:8080/)

4 - Executar testes

```bash
yarn test
```
